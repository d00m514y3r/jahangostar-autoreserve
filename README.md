# About
This is a telegram bot for interacting with jahangostar self-service website which is used in iranian universities

# File Structure
- api: interacting with the website
    - auth: login and authentication
    - menu: creating them `Menu` object
- bot: interacting with telegram bot api
    - filters: interface for managing filters
- db: sqlite3 database management
